# Rocket Business CMS Test

Test project built on **WordPress CMS**.

## Project Description

A small business website developed as a CMS-based project using WordPress with a custom theme.

The website includes:

* Articles section
* Services / Promotions section (Custom Post Type)
* Contact form
* Responsive layout
* Mobile slider for services
* Admin panel content management

## Technologies

* WordPress
* PHP
* HTML / CSS
* JavaScript
* Contact Form 7
* Flamingo
* WP Mail SMTP

## Features

* Custom WordPress theme
* Custom Post Type for services/promotions
* Articles section with dynamic content
* Responsive layout
* Mobile slider for services
* Contact form with email notifications
* Form submissions stored in database (Flamingo)
